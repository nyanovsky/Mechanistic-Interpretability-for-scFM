  Question 2: Steering Coefficient and Normalization                                                                                               
                                                                                                                                                   
  This is critical and often under-discussed. Here's a principled approach:                                                                        
                                                                                                                                                   
  The Scale Problem                                                                                                                                
                                                                                                                                                   
  When you compute:                                                                                                                                
  steering_vector = sum(coeff[i] * decoder[i] for i in steering_features)                                                                          
                                                                                                                                                   
  You need to worry about:                                                                                                                         
  1. Decoder norms vary (you found negative correlation with PR)                                                                                   
  2. Feature activation magnitudes vary                                                                                                            
  3. Original activation magnitudes vary by layer                                                                                                  
  4. Multiple features can compound effects                                                                                                        
                                                                                                                                                   
  Best Practice: Anthropic's Approach (Recommended)                                                                                                
                                                                                                                                                   
  Anthropic scales by typical feature activation magnitudes:                                                                                       
                                                                                                                                                   
  # For each feature, use its natural activation scale                                                                                             
  steering_vector = 0                                                                                                                              
  for feature_id in steering_features:                                                                                                             
      decoder = sae.decoder[feature_id]  # shape: (d_model,)                                                                                       
                                                                                                                                                   
      # Scale by the typical activation magnitude for this feature                                                                                 
      typical_activation = compute_typical_activation(feature_id)                                                                                  
                                                                                                                                                   
      steering_vector += typical_activation * decoder                                                                                              
                                                                                                                                                   
  # Then multiply by steering strength α                                                                                                           
  x_steered = x_reconstructed + alpha * steering_vector + error                                                                                    
                                                                                                                                                   
  Advantages:                                                                                                                                      
  - Uses the natural scale of each feature (how much it typically activates)                                                                       
  - alpha is intuitive: α=1 means "typical activation", α=5 means "5x typical"                                                                     
  - No arbitrary normalization needed                                                                                                              
  - Anthropic tested α ∈ [0, 10] in practice                                                                                                       
                                                                                                                                                   
  Alternative: Normalize by Decoder Norms                                                                                                          
                                                                                                                                                   
  If you don't have activation statistics readily available:                                                                                       
                                                                                                                                                   
  steering_vector = 0                                                                                                                              
  for feature_id in steering_features:                                                                                                             
      decoder = sae.decoder[feature_id]                                                                                                            
                                                                                                                                                   
      # Normalize each decoder to unit norm                                                                                                        
      decoder_normalized = decoder / torch.norm(decoder)                                                                                           
                                                                                                                                                   
      steering_vector += decoder_normalized                                                                                                        
                                                                                                                                                   
  # Normalize total steering vector to match typical activation scale                                                                              
  activation_norm = torch.norm(x_reconstructed, dim=-1, keepdim=True)                                                                              
  steering_normalized = steering_vector / torch.norm(steering_vector) * activation_norm                                                            
                                                                                                                                                   
  # Apply with strength α                                                                                                                          
  x_steered = x_reconstructed + alpha * steering_normalized + error                                                                                
                                                                                                                                                   
  Advantages:                                                                                                                                      
  - Treats all features equally (each contributes 1 unit)                                                                                          
  - Scales steering to match typical activation magnitude                                                                                          
  - Simpler if you don't have feature statistics
